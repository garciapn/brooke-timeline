# DateTrack 🗓️

A beautiful, personal milestone timeline tracker built for tracking multi-phase journeys with date-based goals and windows.

## Features

- **Live counters** — Days, weeks, and months since your start date, updated in real time
- **Timeline ribbon** — Visual phase-by-phase breakdown of your journey
- **Scenario cards** — Multiple what-if scenarios organized by path (Natural, Natural→IVF, IVF Only, etc.)
- **Goal Window tracker** — Animated progress bar showing where you are in your target window
- **Mobile responsive** — Works beautifully on any device

## Tech

- Single `index.html` file — no build step, no framework, no backend
- Vanilla HTML + CSS + JavaScript
- Google Fonts (Playfair Display + Lato)
- Open in any browser

## Live Site

**https://garciapn.github.io/brooke-timeline/**

## Customization

All dates and scenario data are hardcoded in the JS section of `index.html`. Search for `START_DATE` to find the configurable values.
